## Vue d'ensemble du projet

**Pizibot** est un projet personnel de robot mobile autonome développé sous **ROS 2 Humble**. L'objectif principal est d'approfondir mes compétences en navigation robotique tout en créant une plateforme fonctionnelle capable de naviguer de manière autonome dans un environnement domestique.

## Réalisations techniques

### Design et fabrication
- **Conception 3D** complète du châssis et des supports
- **Fabrication** par découpe laser et impression 3D
- **Câblage électronique** et intégration des composants
- **Configuration système** (Ubuntu + ROS 2)

### Architecture logicielle

```yaml
# Stack ROS 2 Pizibot
ros2_packages:
  - ros2_control        # Contrôle hardware abstraction
  - nav2_bringup        # Navigation autonome
  - slam_toolbox        # Cartographie SLAM
  - robot_localization  # Fusion de données
  - tf2_ros             # Transformations
```

### Simulation Gazebo

Le robot est entièrement simulable dans Gazebo, permettant de tester les algorithmes avant déploiement réel :

```bash
# Lancement simulation Pizibot
ros2 launch pizibot_gazebo pizibot_world.launch.py
ros2 launch nav2_bringup navigation_launch.py
```

## Fonctionnalités implémentées

### 1. SLAM et cartographie

Utilisation de **slam_toolbox** pour la création de cartes 2D en temps réel :

```python
# Configuration SLAM
slam_params = {
    'resolution': 0.05,           # 5cm par pixel
    'max_laser_range': 12.0,      # Portée Lidar
    'minimum_travel_distance': 0.5,
    'minimum_travel_heading': 0.5
}
```

### 2. Navigation autonome Nav2

Intégration complète de **Nav2** pour la planification de trajectoires et l'évitement d'obstacles :

- **Planificateur global** : NavFn ou Smac Planner
- **Contrôleur local** : DWB ou MPPI Controller
- **Recovery behaviors** : Spin, backup, wait

### 3. Commande vocale

Le robot peut recevoir des commandes vocales pour naviguer vers des pièces prédéfinies :

```python
# Exemple de commande vocale
"Pizibot, va dans la cuisine"
→ Récupération des coordonnées de la cuisine
→ Envoi du goal à Nav2
→ Navigation autonome avec évitement d'obstacles
```

## Architecture matérielle

| Composant | Description |
|-----------|-------------|
| **Raspberry Pi 4** | Unité de calcul (8GB RAM) |
| **Lidar 2D** | RPLidar A1/A2 pour SLAM |
| **Arduino/ESP32** | Contrôle bas niveau moteurs |
| **Moteurs DC** | Avec encodeurs pour odométrie |
| **Caméra** | Pour vision (futur développement) |
| **Micro** | Reconnaissance vocale |
