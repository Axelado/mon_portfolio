## Contexte et enjeux

Projet développé chez **CI-Drone** pendant la pandémie COVID-19, visant à créer un robot autonome de désinfection pour espaces publics et privés. Réponse rapide aux besoins sanitaires urgents avec contraintes de délai et d'efficacité.

## Cahier des charges

### Exigences fonctionnelles
- **Désinfection automatisée** : Pulvérisation contrôlée de solution désinfectante
- **Navigation autonome** : Déplacement programmé dans zones définies
- **Sécurité sanitaire** : Évitement des personnes et zones sensibles
- **Monitoring** : Suivi temps réel de l'état et des performances

### Contraintes techniques
- **Autonomie** : 6h de fonctionnement continu
- **Capacité** : Réservoir 20L de solution désinfectante
- **Portée** : Pulvérisation efficace sur 3m de rayon
- **Fiabilité** : Fonctionnement 24/7 possible

## Architecture système

### Unité de contrôle
```c
// Code principal Arduino - Boucle de contrôle
void main_control_loop() {
    // Lecture capteurs environnementaux
    distance_sensors_data = read_ultrasonic_array();
    presence_detected = check_human_presence();
    
    // Décision de navigation
    if (!presence_detected && !obstacle_ahead()) {
        move_forward();
        if (should_spray_zone()) {
            activate_spray_system();
        }
    } else {
        avoid_obstacle_or_wait();
    }
    
    // Monitoring système
    send_telemetry_data();
    delay(CONTROL_LOOP_MS);
}
```

### Système hydraulique
- **Pompe péristaltique** : Débit contrôlé et précis
- **Buses de pulvérisation** : Diffusion homogène 360°
- **Capteurs de niveau** : Monitoring réservoir en temps réel
- **Électrovannes** : Contrôle des circuits de distribution

## Modules développés

### 1. Navigation intelligente
```c
// Algorithme d'évitement d'obstacles
void obstacle_avoidance() {
    int left_distance = read_sensor(LEFT_SENSOR);
    int right_distance = read_sensor(RIGHT_SENSOR);
    int front_distance = read_sensor(FRONT_SENSOR);
    
    if (front_distance < SAFE_DISTANCE) {
        if (left_distance > right_distance) {
            turn_left();
        } else {
            turn_right();
        }
    }
}
```

### 2. Système de pulvérisation
- **Contrôle PWM** : Régulation précise du débit
- **Patterns programmables** : Zones de désinfection personnalisables
- **Sécurité** : Arrêt automatique si présence humaine détectée

### 3. Interface utilisateur

- **Caméra** : Vision pour reconnaissance de zones (optionnel)

### Monitoring système
```c
// Surveillance état batterie et réservoir
void system_monitoring() {
    battery_voltage = analogRead(BATTERY_PIN) * VOLTAGE_DIVIDER;
    tank_level = analogRead(TANK_SENSOR_PIN);
    
    if (battery_voltage < LOW_BATTERY_THRESHOLD) {
        trigger_low_battery_alert();
        return_to_charging_station();
    }
    
    if (tank_level < LOW_TANK_THRESHOLD) {
        trigger_refill_alert();
        suspend_spraying_operations();
    }
}
```
